Gambar bukan cuma buat mempercantik tampilan konten. Dalam SEO, gambar bisa membantu menaikkan ranking dan kecepatan loading halaman—kalau kamu tahu cara optimasinya.

Yuk pelajari cara optimasi gambar yang benar tanpa bikin kualitas visualnya jadi buruk.

## 1. Gunakan Format Gambar yang Tepat

- **JPEG**: Cocok untuk foto dan gambar kompleks
- **PNG**: Cocok untuk gambar dengan transparansi atau detail tajam (tapi ukuran besar)
- **WEBP**: Kombinasi keduanya, kualitas oke + ukuran kecil

📌 Tips: Gunakan WebP untuk performa terbaik jika browser pengguna sudah mendukung.

## 2. Kompresi Gambar Sebelum Upload

Gunakan tools seperti:

- [TinyPNG](https://tinypng.com)
- [Squoosh](https://squoosh.app)
- [ImageOptim](https://imageoptim.com) untuk Mac

Tujuan utama: kurangi ukuran file tanpa penurunan kualitas yang berarti.

## 3. Gunakan Nama File yang Deskriptif

Jangan pakai nama `IMG_98732.png`, tapi ganti jadi:

desain-logo-untuk-branding-online.png

Nama file bisa membantu Google memahami isi gambar.

## 4. Tambahkan Alt Text yang Relevan

Alt text penting untuk:

- SEO (Google bisa baca teks, bukan gambar)
- Aksesibilitas (buat pengguna screen reader)

Contoh alt text yang baik:

alt="Contoh desain logo minimalis untuk brand digital"

Jangan isikan keyword secara spammy. Buat alami dan deskriptif.

## 5. Aktifkan Lazy Loading

Lazy loading membuat gambar hanya dimuat saat muncul di layar. Ini bisa:

- Mengurangi waktu loading awal
- Menghemat bandwidth

Di HTML, tinggal tambahkan:

```html
<img src="gambar.jpg" loading="lazy" alt="Deskripsi gambar" />
```
